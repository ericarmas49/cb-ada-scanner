Standalone Accessibility Demo

Scan one page, auto-remediate the frozen snapshot, and compare the before-and-after evidence.

This app checks a single URL against automated WCAG 2.2 AA rules and known best practices. It can suggest AI-generated fixes, but every result still requires manual review before any compliance claim.

Output Before report, after report, preview, artifacts
Scope One settled page snapshot only
Claim Automated checks plus known best practices

The app renders the page in Playwright, captures a frozen HTML snapshot, applies safe fixes, and reruns the same scan on the remediated snapshot.
