Standalone Accessibility Demo

Run a one-page accessibility scan and open the generated report.

This app checks a single URL with the bundled scanner (automated WCAG-oriented rules and heuristics). Results are indicative only; manual review is required before any compliance claim.

Output HTML report, JSON findings, artifacts
Scope One settled page snapshot only
Claim Automated checks plus known best practices

The scanner loads the page in Playwright, runs automated WCAG-oriented checks, and saves a printable HTML report plus JSON under /runs. Use Data demo to load sample results without scanning.
